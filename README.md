STUDY.TRACKER

A personal 12-week coding roadmap tracker built for one purpose — getting from zero Java to hireable by August 31.

No backend. No login. No bullshit. Everything runs in your browser.


What it tracks


12-week roadmap — FCC JavaScript cert, Helsinki Java I & II, React, Spring Boot, microservices, system design, and a deployed capstone
Progress per track — mark each track inside each week as done, independently
LeetCode log — log individual problems with difficulty, topic, week, and notes. ~110 problems across 8 DSA topics
Job applications — kanban pipeline from To Apply → Applied → Followed Up → Interview → Offer/Rejected
Weekly notes — a journal textarea per week, saves automatically
Weekly streak — 12-dot progress indicator, green when all tracks done, amber when partial



Stack


React + TypeScript
Tailwind CSS
localStorage — all data stays in your browser, nothing leaves your machine
Bun



Running locally

bashbun install
bun run dev

Open http://localhost:5173

Data & privacy

All progress, notes, LeetCode logs, and job applications are stored in localStorage in your own browser. Nothing is sent anywhere. Ever.

Clearing browser cache will wipe your data — export or back up periodically if needed.
